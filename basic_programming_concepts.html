Variables, Data Types, and Operators

Variables: Variables are used to store data values. In most programming languages, variables must be declared before they can be used.

Data Types: Data types define the type of data that can be stored in a variable. Common data types include integers, floating-point numbers, strings, and boolean values.

Operators: Operators are symbols that perform operations on operands. Examples include arithmetic operators (+, -, *, /), comparison operators (>, <, ==, !=), and logical operators (&&, ||, !).

Control Structures

If Statement: The if statement allows you to execute a block of code if a specified condition is true.

Else Statement: The else statement allows you to execute a block of code if the same condition is false.

Loops: Loops are used to execute a block of code multiple times. Common types of loops include for loops, while loops, and do-while loops.

Functions and Methods

Functions: Functions are blocks of reusable code that perform a specific task. They can accept input parameters and return output values.

Methods: Methods are functions that are associated with objects. They can modify the object's state or perform operations related to the object.

Data Structures

Arrays: Arrays are used to store multiple values in a single variable. The elements of an array can be accessed using an index.

Lists: Lists are similar to arrays but are more flexible in size and content. They allow for dynamic resizing and support various operations such as adding, removing, and accessing elements.

Dictionaries: Dictionaries, also known as associative arrays or maps, are used to store key-value pairs. Each value is associated with a unique key, allowing for efficient lookup and retrieval.
